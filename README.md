# Spotify-track-info

Node version that generates a `currentSong.txt` that can be used on OBS to show the current spotify song.

# Install

```
npm install
```

# Execute

```
npm start
```

## Template

You can change the `template.txt`

## Template variables

The template suports the followign variables:

- %artist%
- %album%
- %disc_number%
- %duration%
- %played_count%
- %track_number%
- %popularity%
- %id%
- %name%
- %album_artist%
- %artwork_url%
- %spotify_url%
